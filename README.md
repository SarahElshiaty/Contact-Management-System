# Contact Management System (Python)

## Overview

This project is a Python-based Contact Management System designed to manage and organize contact information through a console application.

The system supports contact creation, searching, updating, deletion, validation, file persistence, and automated backup management. The project emphasizes reliable data handling, user-friendly interaction, and structured program design.

---

## Features

### Contact Management
- Add new contacts
- View all contacts
- Search contacts by ID, name, email, or phone number
- Update existing contacts
- Delete contacts
- Save contacts to file

### Data Validation
- Full name validation
- Student ID validation
- Email format validation
- International phone number validation
- Duplicate email prevention
- Duplicate phone number prevention

### Data Management
- Automatic file loading
- Persistent contact storage
- Automatic contact sorting
- Corrupted data detection
- Invalid record filtering

### Reliability Features
- Automatic backup creation before saving
- Update preview before confirmation
- Delete confirmation prompts
- Operation cancellation support
- Input validation and error handling
