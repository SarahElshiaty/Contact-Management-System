# Contact Management System (Python)

## Overview
This project is a Python-based Contact Management System developed to manage and organize contact information through a console application.

The system allows users to add, search, update, delete, and store contacts using file handling techniques and structured data management.

---

## Features
- Add new contacts
- View all contacts
- Search contacts by name
- Update existing contacts
- Delete contacts
- Save contacts to file
- Input validation for email and phone numbers
- File handling using text files

---

## Technologies Used
- Python
- File Handling
- Dictionaries
- Functions
- Input Validation
